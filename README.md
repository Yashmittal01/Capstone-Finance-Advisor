# Financial Portfolio Advisor — Complete AI Project

A fintech-grade AI Advisor using:

- FastAPI Backend
- Streamlit Frontend
- Azure OpenAI (GPT + Embeddings)
- FAISS Vector Store
- Monte Carlo Simulation Engine
- Modular Risk Profiling
- Agentic Architecture

## Running the backend

uvicorn finance_advisor.backend.main:app 

## Runnit frontend 
cd frontend 
streamlit run app.py
